# Cyclistic Bike-Share Usage Analysis

## Overview
This project follows the Google Data Analystics, featured on coursera, "Ask/Prepare/Process/Analyze/Share/Act" framework.
This project analyzes differences in usage patterns between casual riders and annual members to support strategies that could increase annual memberships.

## Business Task
How do annual members and casual riders use Cyclistic bikes differently?

## Data
Cyclistic historical trip data (Dec 2024–Nov 2025).  
Note: This analysis uses trip-level behavioral data only.

## Tools
- Microsoft Excel

## Key Findings
- Casual riders have longer average ride lengths than members
- Members ride more frequently on weekdays
- Casual riders’ usage peaks on weekends
- Members show more consistent usage patterns

## Recommendations (high-level)
- Test messaging that highlights weekday commuting use cases
- Test incentives that encourage weekday usage after weekend rides
- Align marketing intensity with seasonal patterns
